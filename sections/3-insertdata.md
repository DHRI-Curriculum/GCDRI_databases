#Inserting data into a SQL table

1. Insert some data into the table we just created  
	```
	INSERT INTO programs(program) VALUES
	(‘Anthropology’),
	(‘Biology’),
	(‘Linguistics’);
	```

2. Click on "Data" to view the data that you just inserted into the "programs" table.
